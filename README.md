# 💎 FlowState | Glass Aura Pomodoro

A premium, editorial-grade Pomodoro timer and task management application designed for deep work. Featuring a distinctive "Glass Aura" aesthetic with bold typography, dynamic mesh gradients, and immersive lofi vibes.

## ✨ Features

- **⏲️ Glass Aura Pomodoro**: A high-impact countdown timer with atmospheric visual feedback.
- **✅ Integrated Task Tracker**: Manage your "Today's Focus" with a sleek, minimalist list.
- **🎵 Focus Radio**: Built-in YouTube lofi stream to set the perfect mood for concentration.
- **🔔 Smart Notifications**: Browser-level notifications to alert you when sessions or breaks end.
- **⚡ Performance Optimized**: Built with Next.js 15, React 19, and Framer Motion for buttery-smooth interactions.
- **🎨 Premium UI**: Leverages **Syne** and **Plus Jakarta Sans** typography with a deep OLED-black theme.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vishank97/flow-pomo.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Vanilla CSS](https://developer.mozilla.org/en-US/docs/Web/CSS)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Syne & Plus Jakarta Sans via [Google Fonts](https://fonts.google.com/)
