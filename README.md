# ⏱️ FlowState | Pomodoro & Task Tracker

A clean, minimalist productivity tool designed for focused work sessions. 

## ✨ Features

- **⏲️ Pomodoro Timer**: Customizable focus and break intervals to manage your time effectively.
- **✅ Task Tracker**: Keep track of your daily goals with a simple, integrated task list.
- **🎵 Focus Radio**: Built-in lofi music player to create a productive environment.
- **🔔 End-of-Session Alerts**: Audio notifications and desktop alerts to keep you on track.
- **📱 One-Page View**: All tools accessible in a single, non-scrolling interface.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/vishank97/flow-pomo.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **State Management**: React Hooks
- **Styling**: Vanilla CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
